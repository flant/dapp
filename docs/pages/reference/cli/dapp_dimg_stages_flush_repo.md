Удалить кэш приложений проекта в репозитории **REPO**.

```
dapp dimg stages flush repo [options] REPO
```

### Примеры

#### Удалить весь кэш приложений в репозитории localhost:5000/test
```bash
$ dapp dimg stages flush repo localhost:5000/test
```
