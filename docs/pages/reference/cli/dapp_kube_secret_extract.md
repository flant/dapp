Расшифровать данные ключом `DAPP_SECRET_KEY`.

```
dapp kube secret extract [FILE_PATH] [options]
```

### --values
Расшифровать secret-values файл `FILE_PATH`.

### -o OUTPUT_FILE_PATH
Перенаправляет расшифрованные данные в файл `OUTPUT_FILE_PATH`.
