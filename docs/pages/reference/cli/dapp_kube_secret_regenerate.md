Перегенерировать секреты ключом `DAPP_SECRET_KEY`.

```
dapp kube secret regenerate [SECRET_VALUES_FILE_PATH ...] [options]
```

### --old-secret-key KEY
Использовать ключ `KEY` для декодирования.
